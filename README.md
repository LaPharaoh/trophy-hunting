# AchievementForge Landing Page

This is the landing page and community dashboard for AchievementForge — the definitive platform for completionists.

## Tech Stack
- Next.js 16
- Tailwind CSS 4
- Lucide React Icons
- Framer Motion

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
